Flexbox Model

Near the end of this week, we learned something I believe will become very important in the future for creating and styling webpages, flexbox. This layout allows the creator to easily design flexible responsive layout structure without using float or positioning. If you take a look at the the code below, you will see that we have created a class called .flex-container that has display of flex, and direction row. Since it is the direction is in a row, the objects will appear as they are in a row. You can also change the alignment of items within this container and move them around as well. This layout provides a really easy way to design creative websites.

                .flex-container {
                    display: flex;
                    flex-direction: row;
                }
