Primary Keys

In mySQL, primary key represent a single field or combination of fields that uniquely defines a record. They are also generally used to to create a table or alter a table. In order to alter a table, you would the keywords, ALTER and in order to create a table, you would use the key words CREATE. If you look at the code snipet, below you will see an example of how to write SQL and primary keys. For example, in the snipet we used the CREATE keyword in order to create a table named, contacts. In this code we have also created a primary key named contacts_pk that has only one column named, contact_id. In mySQL, you are allowed to have multiple column in a primary. We could also create an ALTER TABLE where we would be able to add, delete, or modify the columns that exists in our tables. I would like to acknowledge Tech on the Net for their assistance on primary keys and also for their code snippet. 

                CREATE TABLE contacts ( 
                    contact_id INT(11) NOT NULL AUTO_INCREMENT,
                    last_name VARCHAR(30) NOT NULL,
                    first_name VARCHAR(25),
                    birthday DATE,
                    CONSTRAINT contacts_pk PRIMARY KEY (contact_id)
                );
